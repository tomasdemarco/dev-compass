# Design System UI

Esta es la librería de componentes de React compartida para todas las aplicaciones web.

## Principios

- **Consistencia:** Todos los componentes deben seguir las guías de estilo de la marca.
- **Accesibilidad:** Los componentes deben cumplir con los estándares WCAG 2.1 AA.
- **Reutilización:** Construidos para ser flexibles y adaptables a diferentes contextos.

## Cómo Instalar

```bash
npm install @nuestra-empresa/design-system-ui
```

## Uso Básico

```jsx
import { Button } from '@nuestra-empresa/design-system-ui';

const App = () => {
  return <Button>Hola Mundo</Button>;
}
```
